# Audio Samples: Impact of Prompt Structure on Emotion Conveyance in AI Music Generation
# 补充材料：提示词结构对 AI 音乐情绪传递的影响研究

This repository contains the AI-generated music samples used in the academic study.
本仓库包含该学术研究中涉及的 AI 生成音乐样本。

**Total Samples / 样本总数：**
There are a total of 16 audio files in this repository (8 generated via uni-dimensional prompts and 8 via multi-dimensional prompts).
本仓库共有 16 个音频文件（包含 8 个单维提示词生成样本和 8 个多维提示词生成样本）。

---
## Naming Convention / 命名规范

The audio files are named using the following format:
所有音频文件均严格采用以下格式进行命名：

`[Emotion]_[Prompt Type]_[Index].mp3`
`[情绪]_[提示词类型]_[编号].mp3`

**Details / 参数说明：**

* **[Emotion] / [情绪]**:
  * `Excited` (兴奋)
  * `Angry` (愤怒)
  * `Depressed` (沮丧)
  * `Relaxed` (放松)

* **[Prompt Type] / [提示词类型]**:
  * `Uni`: Uni-dimensional prompt / 单维提示词
  * `Multi`: Multi-dimensional prompt / 多维提示词

* **[Index] / [编号]**:
  * `01`, `02`: Sample number / 样本序号
